Fix google font

Due to GDPR rulings, you can not let your website load google fonts.

This module hooks into the generation of Odoo's fontend css and replaces the urls of google fonts with relative urls to your webserver.

Additionally your webserver acts as a reverse proxy parsing googles font page and pasing it to the enduser.

If you need this module for a different version of Odoo send a mail to info@bytist.de
